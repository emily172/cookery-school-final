Cookery School Ontology

This site hosts the Cookery School Ontology, developed as part of the Human Centred Systems ontology assignment.

The ontology models a cookery education domain including teachers, students, classes, recipes, techniques, tools, cuisine cultures, and skill levels.

Ontology Files (Linked Data)

  • cookery-school.owl – OWL ontology for semantic modelling and reasoning

How to Reuse the Ontology

The ontology is published as Linked Data and can be reused by:

  • Importing the OWL file into Protégé
  • Extending the model with additional recipes or cuisines
  • Querying the data using SPARQL
  • Using it as a teaching example for OWL and RDF

Access as Linked Data

The ontology can be accessed directly via HTTP at:

https://username.github.io/cookery-school-final/cookery-school.owl

Accessing this URL returns the OWL file, confirming correct publication as machine‑readable Linked Data.
