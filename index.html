Module Human Centered Systems Ontology Assignment

The Cookery School Ontology was developed as part of the Human‑Centred Systems ontology assignment it represents a conceptual ontology for the development of a cookery school.
This cookery school is run by master professionals who have specialised skills in culinary skills, culture, education etc. who are ready to teach cooking enthusiasts to enhance their cooking techniques.
Within this ontology the cookery school is organised based on three core dimensions:

  • Human roles
  • Pedagogical structure
  • Culinary knowledge

By structuring the areas in relation to this discipline, it makes it easy to represent the interactions between teachers, students, classes, recipes, techniques, tools, levels of difficulty and cuisine cultures. This model enables to create a solid design in representing the structure of the cookery school. For this ontology, it is hosted using Linked data hosted by GitPages using GitHub ontology
It represents the main cookery school and query file was was created using Protégé, SPARQL and OWL files.

About the Ontology

Queries to use from queries.owl

The ontology can be queried to retrieve information such as:

Listing out the individuals

This is a test all of the owl files are correctly listed.

        SELECT ?individual
        WHERE {
        ?individual a owl:NamedIndividual .
        }

What recipes are included in a class

Listing out the recipes associated for each class.

            SELECT ?class ?recipe
            WHERE {
    ?class :includesRecipe ?recipe .
        }

Which techniques are required to prepare a recipe

Listing out the recipes with techniques needed for it to be cooked.

        SELECT ?teacher ?class ?recipe
        WHERE {
        ?teacher rdf:type :Teacher ;
           :teaches ?class .
            ?class :includesRecipe ?recipe .
        }
      ORDER BY ?teacher ?class

What each teacher teaches

This query show what type of recipes the teacher is teaching for each class..

        SELECT ?student
        WHERE {
    ?student rdf:type :Student ;
           :attends :class_BeginnerCooking .
        }

Which techniques are required to prepare a recipe

Listing out the recipes with techniques needed for it to be cooked.

        SELECT ?recipe ?technique
      WHERE {
        ?recipe :usesTechnique ?technique .
        }
